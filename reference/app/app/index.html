    compute_metrics ¶

    compute_metrics(
    Y: ndarray, scores_dmd: ndarray, test_size: int
) -> DataFrame

    Compute change-point detection metrics for several detector variants.

    Parameters:

    • Y ¶

      (ndarray) –

      Ground-truth binary labels (1 = change point).

    • scores_dmd ¶

      (ndarray) –

      Online-DMD anomaly scores.

    • test_size ¶

      (int) –

      Window width (in samples) used for time-tolerant scoring.

    Returns:

    • DataFrame –

      DataFrame of F1, FAR, MAR, Delay, TP, FN, FP, and NAB scores

    • DataFrame –

      indexed by detector name.

    progressive_val_predict(
    X: DataFrame,
    U: DataFrame,
    _model: SubIDChangeDetector | Pipeline,
    *,
    compute_alt_scores: bool = False,
    _progress_bar: DeltaGenerator | None = None
) -> tuple[ndarray, dict[str, ndarray]]

    Run prequential (score-then-learn) evaluation over all samples.

    Parameters:

    • X ¶

      (DataFrame) –

      State-variable observations, one row per time step.

    • U ¶

      (DataFrame) –

      Control-variable observations aligned with X.

    • _model ¶

      (SubIDChangeDetector | Pipeline) –

      Fitted or partially-fitted detector or pipeline.

    • compute_alt_scores ¶

      (bool, default: False ) –

      When True, also record the raw DMD distance diff.

    • _progress_bar ¶

      (DeltaGenerator | None, default: None ) –

      Optional Streamlit progress bar updated each step.

    Returns:

    • tuple[ndarray, dict[str, ndarray]] –

      Tuple of (anomaly scores array, metadata dict of auxiliary arrays).
