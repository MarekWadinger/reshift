\section*{Notation}

Throughout this paper, vectors are denoted by lowercase italic letters (e.g., \(a\)), and matrices by uppercase letters (e.g., \(A\)). Time-dependent variables are indexed by a subscript \(k\) to indicate discrete time steps, as in \(\mathbf{x}_k\) for the state vector at time \(k\) and \(\mathbf{u}_k\) for the control input. Augmented matrices are indicated by a bar (e.g., \(\bar{A}_k\)), while variables transformed or projected (e.g., via singular value decomposition or subspace projection) are denoted with a tilde (e.g., \(\tilde{A}\)). The prime symbol (e.g., \(X_k'\)) denotes the time-shifted or successor value of a variable. Sets are represented by calligraphic letters (e.g., \(\mathcal{X}\)), and the set of real numbers is denoted by \(\mathbb{R}\). All vectors are assumed to be column vectors unless otherwise specified. Domain-specific variables, such as Koopman operator-related quantities, follow the same conventions, with context provided as needed. The symbols are summarized in Table~\ref{tab:notation}.
